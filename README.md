# 2026 夏季地球科學教師研習課程

> 🌍 **地震科學 × 地球物理 × AI/Agent 工具輔助教學**
> 📅 **時間**：2026/7/17 起，每週四上午，共 6 週（每次 4 小時）
> 👩‍🏫 **對象**：台北市國民小學現職教師
> 🎯 **目標**：將地震科學轉化為小學可理解的教材，並結合現代工具輔助教學

---

## 📌 專案簡介

本專案是 **2026 夏季地球科學教師研習課程** 的教材與資源倉庫，專為台北市國小教師設計。課程結合 **地震科學、地球物理、AI/Agent 工具**，目標是讓教師能夠：

1. **建立地震科學的基本理解**（地震學、地震防災、地球物理）。
2. **實作簡易地震監測與教學設計**（如手機感測器、IRIS 資料分析）。
3. **熟悉現代教學工具**（如 Google GAS、GitHub、Hugging Face、Vercel、AI Agent 等）。
4. **設計適合小學生的地震科學教案**，並帶回學校應用。

## 🎯 如何開始

### 1. **閱讀課程計畫**
- 詳細了解每週的主題、工具、作業要求，請參閱 [course_plan.md](course_plan.md)。

### 2. **準備工具**
- 註冊並設定所有必要的工具（Google、GitHub、Hugging Face、Vercel 等）。
- 安裝本地開發環境（Git、VS Code、Python 等）。

### 3. **跟隨學習指引**
- 參考 [學習指引.md](學習指引.md)，了解如何高效學習這門課程。

### 4. **動手實作**
- 每週完成 1 件可評量的作業，並將成果提交到 GitHub 或其他平台。

## 📢 如何貢獻

如果你是課程的參與者或貢獻者，歡迎：
1. **提交作業**：將每週的作業上傳到 GitHub，並確保格式和內容符合要求。
2. **分享教案**：如果你設計了優秀的教案或工具，可以提交 Pull Request 到本倉庫。
3. **回報問題**：如果發現課程資料或工具有問題，請開 Issue 回報。
4. **改進建議**：對課程內容有建議，歡迎提出討論。
