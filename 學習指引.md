### 📅 **Week 2：地震儀、加速度、地震動與震級**
**主題**：地震監測的核心問題：地震到底記錄了什麼？地表振動如何被量化？
**工具**：手機感測器、Google Sheets、GAS、IRIS ground-shaking activity

#### 學習目標
- 了解地震監測的基本原理。
- 使用手機感測器或專業儀器記錄地震動。
- 理解加速度、速度、位移等地震參數。

#### 學習策略
1. **理論學習**：
   - 了解地震動的基本概念（PGA、PGV、加速度、速度、位移）。
   - 了解震級（Magnitude）和地震動強度（Intensity）的區別。
2. **實作練習**：
   - 使用手機感測器或專業地震儀記錄地震動。
   - 使用 Google Sheets 分析地震動資料，繪製加速度-時間圖。
   - 使用 GAS 自動化資料收集和分析。
3. **IRIS 資源**：
   - 使用 IRIS 的 ground-shaking 活動，模擬地震動記錄。
4. **作業準備**：
   - **作業 2**：完成簡易地震監測活動。
   - 提交一份地震動記錄和分析報告（包含圖表和說明）。

#### 額外資源
- [IRIS Ground Motion Visualizations](https://www.iris.edu/hq/retm)
- [手機地震儀 App（如 iSeismometer）](https://apps.apple.com/us/app/iseismometer/id291273302)

### 📅 **Week 3：P-S 到時差、旅行時間曲線與震源定位**
**主題**：地震科學的基本問題：只靠幾個測站的地震記錄，如何定位地震發生在哪裡？
**工具**：IRIS 震源定位活動、Python、opencode、GitHub

#### 學習目標
- 理解 P 波和 S 波的特性。
- 學習如何使用 P-S 到時差計算震源距離。
- 了解旅行時間曲線的原理。
- 完成震源定位的教學範例。

#### 學習策略
1. **理論學習**：
   - 了解 P 波（縱波）和 S 波（橫波）的傳播速度和特性。
   - 了解旅行時間曲線（Travel Time Curve）的概念。
2. **實作練習**：
   - 使用 IRIS 的震源定位活動，模擬 3 個測站的地震記錄。
   - 手動計算 P-S 到時差，並推算震源距離。
   - 使用 Python 或 Google Sheets 繪製旅行時間曲線。
3. **教學設計**：
   - 設計一個簡單的震源定位教學活動，適合小學生理解。
4. **作業準備**：
   - **作業 3**：完成震源定位教學範例。
   - 提交 3 個測站的到時差計算和震源定位結果。

#### 額外資源
- [IRIS Earthquake Location Activity](https://www.iris.edu/hq/retm/lessons/locating_an_earthquake)
- [P-S 到時差計算工具](https://www.iris.edu/hq/retm/tools/travel_time)

### 📅 **Week 5：規模、震度與建築物影響**
**主題**：地震防災教學中最困擾的問題：地震規模和震度有什麼不同？為什麼同一個地震，不同地點搖晃程度不同？為什麼同一棟建築物，地震來臨時感受不同？
**工具**：IRIS Magnitude/Intensity、Buildings in Earthquakes、Vercel、GAS

#### 學習目標
- 理解地震規模（Magnitude）和震度（Intensity）的區別。
- 了解建築物在地震中的反應和損壞原因。
- 設計地震防災互動網頁。

#### 學習策略
1. **理論學習**：
   - 了解地震規模（如里氏規模、矩規模）和震度（如修訂麥加利震度）的定義和計算方式。
   - 了解建築物的共振現象和抗震設計。
2. **資料分析**：
   - 使用 IRIS 的 Magnitude/Intensity 資源，比較不同地震的規模和震度分布。
   - 分析建築物在不同地震中的損壞案例。
3. **網頁設計**：
   - 使用 Vercel 部署一個簡單的地震防災互動網頁。
   - 使用 GAS 設計後端邏輯，如地震資料查詢或防災知識測驗。
4. **作業準備**：
   - **作業 5**：完成地震防災互動網頁初版。
   - 提交一份包含規模 vs 震度比較圖、建築物影響說明的報告。

#### 額外資源
- [IRIS Magnitude vs. Intensity](https://www.iris.edu/hq/retm/faq/magnitude_vs_intensity)
- [Vercel 入門教學](https://vercel.com/docs)
